# SecureNet Guardian 2025

برنامج متكامل لفحص الشبكات والأجهزة المتصلة بالراوتر، مع كشف البورتات المفتوحة وأنظمة التشغيل، مع وضع مراقبة مستمرة (SOC Mode) ونسخ النتائج بسهولة.

---

## المتطلبات

- **Python 3.8 أو أحدث**
- [Nmap](https://nmap.org/download.html) مثبت على جهازك (عادة في أحد المسارات التالية):
  - `C:\Program Files (x86)\Nmap\nmap.exe`
  - `C:\Program Files\Nmap\nmap.exe`
- مكتبة python-nmap:
  ```bash
  pip install python-nmap
